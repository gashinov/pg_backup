# pg_backup
Набор скриптов в формате bat для автоматизации резервного копирования и восстановления баз PostgreSQL сервера 1С на Windows Server

pg_dump.bat - скрипт резервного копирования с помощью утилиты pg_dump. Формат копии - directory. Параллелизм установить согласно своим системным данным.
Установить исполнение планировщиком задач по таймеру. Прописать свои переменные для выполнения резервного копирования баз, перечисленных в файле pg_dump.ini
Выполнение копирования не требует отключения клиентов, но обязательно отсутствие блокировки таблицы конфигурации Конфигуратором.

Таблица config сохраняется отдельно по причине превышения размера таблицы в нестандартных конфигурациях возможностям stdio PostgreSQL сервера. Аналогично можно резервировать другие "дутые" таблицы.

pg_dump.ini - файл с перечислением баз, подлежащих резервному копированию.
Имя каждой базы должно находиться в отдельной строке.

pg_dump.log - файл отчета по работе скрипта резервного копирования

pg_restore.bat - скрипт восстановления базы данных из копии, выполненной скриптом резервного копирования.
